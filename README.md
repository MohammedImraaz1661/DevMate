DevMate – Your Friendly Event Partner  

Simplify college event management with a sprinkle of dev magic.

DevMate is a full-stack web app built to make college event registration and management quick, smart, and frustration-free.  
No more “Google Form lost again?” chaos — DevMate keeps everything structured, responsive, and ready to deploy.

---

Features
- Easy event creation and registration  
- QR-based registration system (coming soon!)  
- Secure data storage with MongoDB  
- Fast, responsive, and animated UI  
- Smooth feedback messages and error handling  
- Built and refined with real-world hackathon pain  

---

Tech Stack

Frontend: React.js + Tailwind CSS + Framer Motion  
Backend: Node.js + Express.js + MongoDB (Mongoose)  
Runtime: Vite  
Extras: Axios, Dotenv, Toastify, Spinner Animations
